INSTRUCTION

In each round of this game, you need to guess a word or phrase based on its topic (as a hint) by selecting letters. For every word or phrase, you have 6 lives corresponding to 6 attempts to guess. If the number of lives is down to 0 in any round, the game is over.

The game UI contains following parts, as seen in the figure below:

  1. Quit: If you want to quit the game, please use this button.
  2. Lives: The number of lives left. If it is smaller than 3, it will be red color to indicate the player has low lives.
  3. Round: The current playing round.
  4. Topic: The topic that associated to the hidden word.
  5. Figure: The image representation of the lives.
  6. Hidden word: Only English alphabet characters are hidden as underscores. If you correctly guesses a letter, it will be shown here.
  7. Game status: Each time you guess a letter, when you solve a word, or when the game is over, a textual status will be shown here.

    In case that the game is over, you need to enter your name to save your result.

  8. Letter buttons: This is where you guess the letters in a word. The gray buttons are the previously selected letters.
